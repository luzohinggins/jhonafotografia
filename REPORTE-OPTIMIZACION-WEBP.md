## 📝 INSTRUCCIONES PARA ACTUALIZAR EL HTML

Necesitas reemplazar las referencias de imágenes en el archivo `index.html` y otros archivos HTML según los siguientes criterios:

### 🔧 Cambios necesarios en cada tipo de referencia:

#### 1️⃣ **Etiquetas `<img>`**
```html
<!-- ANTES -->
<img src="ruta/imagen.jpg" alt="descripción">

<!-- DESPUÉS -->
<img src="ruta/imagen.webp" alt="descripción">
```

#### 2️⃣ **Etiquetas `<picture>` (Recomendado para soporte retro)**
```html
<!-- OPCIÓN 1: Soporte WebP con fallback -->
<picture>
  <source srcset="ruta/imagen.webp" type="image/webp">
  <img src="ruta/imagen.jpg" alt="descripción">
</picture>

<!-- OPCIÓN 2: Solo WebP (ya que todos los navegadores modernos lo soportan) -->
<img src="ruta/imagen.webp" alt="descripción">
```

#### 3️⃣ **Imágenes en CSS (background-image)**
```css
/* ANTES */
background-image: url('ruta/imagen.jpg');

/* DESPUÉS */
background-image: url('ruta/imagen.webp');
```

#### 4️⃣ **Favicon y Apple Touch Icon**
```html
<!-- ANTES -->
<link rel="apple-touch-icon" href="apple-touch-icon.png">
<link rel="icon" type="image/x-icon" href="favicon.ico">

<!-- DESPUÉS -->
<link rel="apple-touch-icon" href="apple-touch-icon.webp">
<link rel="icon" type="image/webp" href="favicon.webp">
```

#### 5️⃣ **Web App Manifest**
```json
{
  "icons": [
    {
      "src": "web-app-manifest-192x192.webp",
      "sizes": "192x192",
      "type": "image/webp"
    },
    {
      "src": "web-app-manifest-512x512.webp",
      "sizes": "512x512",
      "type": "image/webp"
    }
  ]
}
```
